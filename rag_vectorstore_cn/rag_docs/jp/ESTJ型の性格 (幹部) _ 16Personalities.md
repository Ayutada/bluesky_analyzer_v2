# 幹部

> 秩序はあらゆるものの基礎である。
>
> エドマンド・バーク

幹部は伝統や秩序を非常に大事にする人たちで、社会的に容認されている事柄や善悪についての自らの理解をもとに家族やコミュニティを団結させます。正直である、何かに専念する、尊厳を保つ——このような価値観を大切にしながら分かりやすいアドバイス・指導をするので人々にありがたがられているでしょう。困難な状況にあっても喜んで皆を先導するタイプでもあります。人をまとめる能力に誇りを持っているので頻繁にコミュニティのまとめ役になり、皆に愛されている地元のイベントを祝うために多くの人を呼び集めたり、家族やコミュニティの団結のために重要な伝統的価値観を擁護したり頑張る人たちです。

![幹部 (ESTJ)型の性格](https://www.16personalities.com/static/images/personality-types/scenes/sentinels_Executive_ESTJ_introduction.svg)

## 手本を示す

このようなリーダーシップは民主的な社会で大いに重宝されている上に幹部型は人口の11%以上を占めるので、多くの大統領が幹部型であることは納得でしょう。実力に裏付けられた権力および法の支配を強く信じている人たちでもあります。献身的に何かに取り組む、意図的に正直である、特に仕事において怠けたりズルをしたりすることを完全に拒否する——これらの価値観を自ら行動で表すことで、周りの人に手本を示しながら指導するでしょう。きつい肉体労働は人格形成に最適だ…と主張するタイプの人たちです。

幹部は自分の周囲の環境をよく把握できていて、検証可能で明確な事実からなる世界に生きています。自分の知識を疑わないので、大きな抵抗があっても自分の道義を貫き、容認できること・できないことについての持ち前の明確なビジョンを推し進めるでしょう。幹部の意見は机上の空論ではないので、極めて難しいプロジェクトにも進んで関わり、アクションプランを改善したり、仕事を進めながら詳細を整理したりして、ものすごく複雑なタスクであっても簡単でとっつきやすいものにしてくれるでしょう。

それでも、幹部は一人で仕事をするわけではないので、周りの人も自分同様高い勤労意欲を持ち、信頼できる人であるよう求めます。約束を守る人たちでもあります。パートナーや部下が怠惰・無能であったり、またはもっとひどい場合、嘘をついたりした結果、共同作業が台無しになった——そんな場合には激怒し、自分の怒りを隠しもしないでしょう。よって、“柔軟性がない”という悪評がついてしまうこともあります（これは番人グループに属するすべての性格タイプが持つ気質です）。でも、幹部の頑固さは気まぐれによるものではなく、「社会がうまく機能するには自分が大切にする価値観は不可欠だ」という考えに基づくものなのです。

## 大きな責任を負う

幹部は“理想的な市民”の典型です。近所の人に手を貸し、法律を擁護し、自分が大切にするコミュニティや組織に皆が参加するよう努めるでしょう。

全員が同じ道をたどるとは限らないし、同じやり方で貢献するとも限らない——これを認識するのが、幹部にとって主な課題です。真のリーダーは個人とグループ双方の強みを理解しながらも、各個人の意見を検討するよう励みます。そうすれば確実にすべての事実を集めることができ、皆の利益になるように集団を率いることができるでしょう。
